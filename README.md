# Visual-X
A Synapse X Custom UI.


# Installation

1. Download VisualX.rar from the releases
2. Extract the downloaded rar in your Synapse-X directory.
3. You're ready to go!

In the end, your Synapse-X directory should look like this:

Just check for updates by clicking the "Changelog/Update" button.
